## E-Commerce Web Application (dj_project)

A full-stack e-commerce web application built end-to-end as a personal project to learn and apply modern web development practices.

###Live Demo
**https://donovanearl-1.onrender.com**

###Tech Stack
| Layer | Technologies |
|-------|-------------|
| Backend | Python, Django, Django REST Framework (DRF) |
| Frontend | React, Vite, JavaScript (ES6+), CSS |
| Database | PostgreSQL |
| Payments | Stripe API |
| Deployment | Render (PaaS) |
| Tools | Git, VS Code, Postman |

###Features
- Product catalogue with browsing and search
- Shopping cart management
- User registration and authentication
- Stripe payment checkout integration
- Responsive UI built with React and CSS
- RESTful API backend serving the React frontend
- State management using React Hooks (useState, useEffect)
- Axios for frontend-to-backend API calls

### Project Structure
```
dj_project/
├── backend/          # Django REST Framework API
│   ├── products/     # Product models, views, serializers
│   ├── orders/       # Cart and order management
│   └── users/        # Authentication
└── frontend/         # React + Vite
    └── src/
        ├── components/
        └── pages/
```
